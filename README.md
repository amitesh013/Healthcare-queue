# 🏥 Healthcare Queue Management System

A full-stack healthcare queue management system designed to streamline patient appointments, emergency handling, doctor availability, and report generation. The system helps hospitals reduce waiting times and improve patient flow through smart queue prioritization.

## 🚀 Features

* 👤 Patient Registration & Login
* 🩺 Doctor & Staff Authentication
* 📅 Appointment Booking System
* 🚨 Emergency Patient Prioritization
* 🎟️ Automatic Token Generation
* ⏳ Smart Queue Management
* 📍 Patient Status Tracking
* 📄 Medical Report PDF Generation
* 💳 Payment Receipt Generation
* 👨‍⚕️ Doctor Attendance Management
* 🔄 Real-Time Queue Updates

## 🏗️ System Workflow

1. Patient registers and logs in.
2. Appointment is scheduled with a doctor.
3. Token number is automatically assigned.
4. Emergency cases are prioritized in the queue.
5. Staff updates patient status (Arrived, In Consultation, Completed).
6. Doctors generate medical reports.
7. Payment receipts are generated automatically.
8. Completed appointments are removed from the queue.
